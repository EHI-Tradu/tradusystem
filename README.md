# TraduSystem Website

Static bilingual website designed for GitHub Pages.

## Publish on GitHub Pages
1. Create a new public repository, for example `tradusystem-site`.
2. Upload all files and folders in this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then Save.
6. GitHub will provide a temporary website address.

## Connect tradusystem.com
The included `CNAME` file tells GitHub Pages to use `tradusystem.com`. DNS records must also be changed at the domain registrar. Confirm the registrar and current DNS settings before changing them.

## Editing contact information
Open `index.html` and search for `tradusystem@gmail.com`, phone numbers, or other text.
